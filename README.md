# Camera-IMU Calibration Sensitivity and Recovery Benchmark

## Summary

This repo measures how controlled camera-IMU calibration perturbations affect OpenVINS VIO accuracy on EuRoC MAV.

The MVP studies camera-IMU rotation and translation perturbations, evaluates trajectories with evo, records ATE/RPE/failure status in results/metrics.csv, and summarizes results in an error-budget report.

No benchmark results are claimed until generated by scripts.
